# GTA Online Heist Replay Script

This script blocks Rockstar Games’ cloud services from saving your game state to their servers. 
By preventing auto-saves at key moments, it allows you to replay heist finals repeatedly without re-doing the setup preps.

Tested & Confirmed Working:
- Cayo Perico Heist ✅
- The Kortz Center Heist ✅
- Diamond Casino Heist ❓
- Doomsday Heist ❓
- Original Heists ❓

# How to use it
Example: Cayo Perico but it's almost the same in every heist.
NOTE !!!: The script needs timing. You need to find the perfect timing for each heist to activate the script.

1. Play Normally: Play through the heist as you normally would.
2. Activate Script: Once you enter the compound cutscene, press 1 to activate the script.
3. Finish the Heist: Complete the rest of the heist normally.
4. Wait for Message: Upon completion, wait until you see the notification: "SAVING FAILED: The Rockstar cloud servers..."
5. Exit to Story Mode: Once that pop-up appears, immediately quit to Story Mode.
6. Deactivate Script: In Story Mode, press 2 to turn off the script, then rejoin GTA Online.
7. Save Progress: To force a manual save afterwards, simply change your outfit via the Interaction Menu.

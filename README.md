# GTA Online Heist Replay Script

This script blocks Rockstar Games’ cloud services from saving your game state to their servers. 
By preventing auto-saves at key moments, it allows you to replay heist finals repeatedly without re-doing the setup preps.

Tested & Confirmed Working:
- Cayo Perico Heist ✅
- The Kortz Center Heist ✅
- Diamond Casino Heist ❓
- Doomsday Heist ❓
- Original Heists ❓
